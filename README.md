# I Can Solder - DIP
Very easy soldering kit for kids and beginners.

<img src="images/front.png" height=400px alt="I Can Solder DIP"> <img src="images/back.png" height=400px alt="I Can Solder DIP">

- Status: **Complete**
- Difficulty: **1/5**

| Amount | Name         | Description                        |

| 1      |              | Battery holder                     |
